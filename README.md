# Flixx Movie App

A movie and TV show discovery app built with vanilla JavaScript using the TMDB API. Browse popular movies and shows, view detailed information, and search for specific titles with paginated results.

## Features

- Browse popular movies and TV shows
- View detailed info including ratings, genres, budget, runtime, and production companies
- Search movies or shows with paginated results
- Now Playing slider powered by Swiper.js
- Dynamic backdrop images on detail pages

## JavaScript Concepts Used

- Async/Await & Fetch API
- DOM Manipulation
- Functions, Scope & Execution Context
- Loops & Iteration
- Logic & Control Flow
- Events & Event Delegation
- URL/Query String Parsing
